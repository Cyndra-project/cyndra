# Resources

## Managed resources

The list of managed resources for cyndra is always growing.
If you feel we are missing a resource you would like, then feel to create a feature request for your desired resource.

## Writing your own managed resources

Creating your own resources is actually easy.
You only need to implement the [`ResourceBuilder<T>`](https://docs.rs/cyndra-service/latest/cyndra_service/trait.ResourceBuilder.html) trait for your resource.
